BonBouquins pour les développeurs

Pour accéder à notre API, envoyez une requette sur l'url suivant : https://bonbouquins.com/api/[ISBN] avec [ISBN] l'ISBN du livre de votre choix.

L'API retourne un fichier de json de la forme suivante :

  • author : l'auteur du livre
  • title : le titre du livre
  • year : l'année de publication du livre
  • ISBN : l'ISBN du livre
  • review_count : le nombre d'évaluations sur notre site
  • average_score : la note moyenne sur notre site (sur 5)
